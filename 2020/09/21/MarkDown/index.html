    MarkDown 是一种轻量级标记语言，它允许我们使用易读易写的纯文本格式编写文档，目前许多网站都在广泛使用MarkDown来撰写文章或是在论坛发表文章，当然用Hexo+github搭建的个人博客中也支持使用MarkDown来撰写文章，为了更好地搭建个人博客，我们有必要使用MarkDown来帮助我们来编写格式更好的文章。

    一.标题

    标题使用#号标记

    一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。

    这是一级标题

    这是二级标题

    这是三级标题

    注：#后面要有空格，不然无法正确显示

    二.段落格式

    MarkDown可以使用三种字体：斜体、加粗、斜体加粗。

    斜体：在文字两端各加一个*

    这是斜体

    粗体：在文字两端各加两个*

    这是粗体

    斜体加粗：在文字两端各加三个***

    这是斜体加粗

    分割线使用 — 来建立分割线

    文字两端使用 ～～ 来使文字变成删除格式

    这是文字的删除格式

    注：～～必须是英文的～

    下划线是通过html的 u 标签来实现的，在文字两端加上即可

    这是下划线

    三.列表

    列表分为 有序列表 和 无序列表

    无序列表使用 *、+、- 作为列表标记

    • 第一项
    • 第二项
    • 第三项

    有序列表使用 1. 2. 3.标记

    1.第一项
    2.第二项
    3.第三项

    列表嵌套

    第二层为第一层换行后加一个tab

    • 第一层
      • 第二层
        • 第三层
          • 第四层

    区块引用和嵌套

    区块引用：在段落开头使用 > 符号，后面跟一个空格

    这是区块一

    这是区块二

    区块嵌套：一个>是最外层，两个是第一层嵌套，以此类推

    父亲

    儿子

    孙子

    注：区块嵌套中>也要隔一个空格

    四.链接和图片

    链接使用：链接名称

    如：zleo的博客

    图片使用：图片的使用和链接使用一样

    如：菜鸟教程

    若使用本地图片，则链接为 在与本md文件同名的文件夹中的图片名称，本文的文件名为MarkDown，则链接为MarkDown图片的名称

    MarkDown还不支持指定图片的高度和大小，若有需要可以使用html的img标签

    五.表格

    MarkDown中用|分隔不同单元格，用-来分隔表头和其他行

    如：

    表头  表头  表头
    内容 内容 内容

    表格中默认为向左看齐

    向右看齐：右边加

    文字居中：两边加

    注：&emsp; 表示两个空格

    六.代码块

    如果是段落上的一个函数或者是片段代码可以用反引号包起来

    如：printf() 函数

    如果是一个代码块，则全选代码块加一个tab即可

    #include<stdio.h>
#include<iostream>
using namespace std;

    以上就是MarkDown的基本语法。
